# DATA-006：NASA Harvest 农业监测


素材类型：机构/遥感平台  
核心信息：NASA Harvest 是 NASA 全球粮食供应与农业联盟，利用地球观测数据和公私合作提供及时、可行动的农业洞察，支持全球和美国农业韧性与决策；其重点包括政策快速评估、市场/贸易/供应链、农业风险、农业保险、早期预警、AI/ML 和数据系统。  
可用角度：

- 遥感 + AI 可服务政策、保险、贸易和早期预警，不只服务单个农场。
- 国际农业大数据项目可以参考 NASA Harvest 的“数据产品 + 决策支持 + 合作网络”模式。

来源：NASA Harvest，https://www.nasaharvest.org/

# DATA-005：Landsat 长时间序列数据


素材类型：公开数据源/遥感  
核心信息：NASA/USGS Landsat 从 1972 年以来持续获取地表影像，是最长连续空间地表观测记录之一；NASA 表示所有 NASA 数据免费开放，Landsat 可用于作物周期变化、森林变化、城市扩张等长期变化研究。  
农业用途：

- 长周期地表变化分析。
- 历史耕地变化。
- 农业用水监测。
- 干旱与灾害历史复盘。
- 与 Sentinel-2 互补做长期基线。

来源：NASA Landsat，https://science.nasa.gov/mission/landsat/

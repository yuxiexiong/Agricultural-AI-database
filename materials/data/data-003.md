# DATA-003：农业数据共享的信任问题


素材类型：数据主权/信任  
核心信息：农业数据共享常因农户担心数据被滥用、平台锁定、收益分配不清而受阻。AgriTrust 等研究把这种问题概括为农业数据价值被信任与互操作不足锁在孤岛中。  
可用角度：

- 数据共享不是技术接口问题，而是信任、合约、收益分配和合规问题。
- 项目设计中要给出数据授权、用途限制、价值回馈和审计机制。

来源：AgriTrust: a Federated Semantic Governance Framework for Trusted Agricultural Data Sharing，https://arxiv.org/abs/2511.05572

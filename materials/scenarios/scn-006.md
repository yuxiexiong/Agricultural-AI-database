# SCN-006：农技知识助手


适合客户：农业服务公司、园区、地方产业平台、农资企业。  
数据需求：知识库、作物 SOP、政策、FAQ、农药肥料规则、当地天气与地块数据。  
模型路线：LLM + RAG + 知识图谱 + 工具调用 + 安全规则。  
输出：问答、建议、报告、任务提醒。  
验收指标：命中率、专家审核通过率、用户留存、人工服务节省。

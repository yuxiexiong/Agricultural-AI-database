# SCN-002：病虫害识别与预警


适合客户：果蔬、经济作物、植保服务、合作社。  
数据需求：图片、天气、历史病害、作物阶段、地区病虫害历。  
模型路线：CV + 风险模型 + 专家规则 + LLM 解释。  
输出：疑似病虫害、置信度、处置建议、是否需专家确认。  
验收指标：识别准确率、提前预警天数、误报/漏报、专家审核通过率。

# CASE-004：NASA Harvest


素材类型：国际公共数据案例  
核心信息：NASA Harvest 利用地球观测数据、AI/ML、数据系统和伙伴网络，服务农业政策快速评估、市场贸易、农业保险、风险评估和早期预警。  
可讲结构：

1. 数据：卫星地球观测。
2. 方法：AI/ML + 数据系统。
3. 服务对象：政策制定者、市场、保险、农户和供应链。
4. 价值：提升粮食安全和农业韧性。

适合话术：遥感农业的最高价值，不只是看田长得好不好，而是在粮食安全、贸易、保险和灾害响应中提供提前量。  
来源：NASA Harvest，https://www.nasaharvest.org/

# STD-003：AgGateway ADAPT


素材类型：精准农业数据互操作  
核心信息：ADAPT 全称 Agricultural Data Application Programming Toolkit，目标是解决精准农业数据在不同软硬件系统之间难以互通的问题；其网站指出多个系统采集/分析田间作业数据，但行业一直难以让不同系统用共享数据格式“对话”。ADAPT Standard 第一版于 2024 年 6 月发布。  
可用角度：

- 用于说明甲方平台要避免厂商锁定，需提前设计数据模型和转换插件。
- 适合做“农业数据普通话”的标准案例。

来源：ADAPT Framework，https://adaptframework.org/

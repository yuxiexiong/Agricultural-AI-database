# Reports Layer

本目录保存交付物模板和报告大纲。它回答的问题是：如何把 database 和 materials 组合成客户可读的交付物。

## 当前规划

| 文件 | 用途 |
|---|---|
| `consulting_report_outline.md` | 农业 AI 技术咨询报告大纲 |
| `monthly_intelligence_template.md` | 月度产业情报简报模板 |
| `implementation_plan_template.md` | 技术落地实施方案模板 |

## 维护规则

- 报告模板不直接堆积来源材料。
- 引用观点时应回链到 `database/knowledge/claims.yml`。
- 引用案例时应回链到 `database/resources/cases.yml`。
- 引用来源时应回链到 `database/resources/` 下的 `RES-XXXX`。
